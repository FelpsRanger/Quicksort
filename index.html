📚 Introdução ao Quicksort

Criado por: Tony Hoare em 1960

Princípio: Dividir para conquistar

Aplicações: Ordenação eficiente de grandes volumes de dados

🎯 Conceito Principal

O Quicksort é como dividir uma pizza gigante em fatias menores até ficar fácil de comer!

🔹 Escolhe um elemento como "pivô"

🔹 Reorganiza o array: elementos menores à esquerda, maiores à direita

🔹 Aplica recursivamente aos sub-arrays

🎲 Gerar Array Personalizado

💻 Implementação do Quicksort

🔧 Código Principal

void quicksort(int arr[], int baixo, int alto) {
    if (baixo < alto) {
        // Particiona o array e obtém o índice do pivô
        int pi = particionar(arr, baixo, alto);
        
        // Ordena recursivamente os elementos antes e depois do pivô
        quicksort(arr, baixo, pi - 1);
        quicksort(arr, pi + 1, alto);
    }
}

int particionar(int arr[], int baixo, int alto) {
    int pivo = arr[alto];    // Escolhe o último elemento como pivô
    int i = (baixo - 1);     // Índice do menor elemento
    
    for (int j = baixo; j <= alto - 1; j++) {
        if (arr[j] <= pivo) {
            i++;
            trocar(&arr[i], &arr[j]);
        }
    }
    trocar(&arr[i + 1], &arr[alto]);
    return (i + 1);
}

🎯 Quando Usar Quicksort?

✅ Use quando:

  • Dados são aleatórios ou semi-aleatórios
  • Memória é limitada (ordena in-place)
  • Performance média é mais importante que garantias
  • Implementação simples é desejada

❌ Evite quando:

  • Dados já estão ordenados (sem otimização)
  • Estabilidade é crucial
  • Sistema não pode ter O(n²)
  • Arrays extremamente grandes (stack overflow)
